# Arbol de proyecto

## Objetivo de aprendizaje
- Gestionar archivos y carpetas del proyecto desde el arbol.

## Sintaxis clave
```text
F2, Delete, Ctrl+C, Ctrl+X, Ctrl+V, Ctrl+D, Esc
```

## Ejemplos
- Crear archivos/carpetas.
- Renombrar, duplicar, mover y eliminar.
- Copiar ruta relativa/absoluta.
- Ejecutar/compilar archivos `.lucia` seleccionados.

## Errores comunes
- Aplicar run/compile a rutas que no son `.lucia`.
- Olvidar limpiar estado pendiente de cortar/copiar.

## Practica sugerida
- Reorganiza un mini proyecto usando duplicar + mover + renombrar.
