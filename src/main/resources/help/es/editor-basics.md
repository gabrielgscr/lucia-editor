# Fundamentos del editor

## Objetivo de aprendizaje
- Usar flujos base del editor con eficiencia en clase y practica.

## Sintaxis clave
```text
Ctrl+S, Ctrl+Shift+S, Ctrl+W, Ctrl+F, Ctrl+H, Ctrl+Shift+F
```

## Ejemplos
- Edicion multi-pestana.
- Busqueda local y global.
- Ir a definicion (`F12`) y buscar referencias (`Shift+F12`).

## Errores comunes
- Olvidar guardar antes de ejecutar/compilar.
- Reemplazo global sin revisar resultados.

## Practica sugerida
- Abre 3 archivos, navega simbolos y ejecuta una busqueda global.
