# Gestor de snippets

## Objetivo de aprendizaje
- Crear plantillas reutilizables para patrones Lucia frecuentes.

## Sintaxis clave
```text
Ctrl+Shift+I -> Insertar snippet
Herramientas -> Gestionar snippets
```

## Ejemplos
- Crear snippet para clase con constructor.
- Crear snippet para bloque try/catch/finally.
- Mantener templates cortos y didacticos.

## Errores comunes
- Snippets demasiado largos que ocultan fundamentos.
- Colisiones de prefijo entre snippets.

## Practica sugerida
- Crea 3 snippets para tu siguiente leccion y prueba autocomplete.
