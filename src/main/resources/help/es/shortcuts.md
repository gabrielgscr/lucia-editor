# Atajos de teclado

## Objetivo de aprendizaje
- Memorizar atajos de alto impacto para productividad.

## Sintaxis clave
```text
F1 Ayuda contextual | Shift+F1 Centro de ayuda | F5 Run | F6 Compile
```

## Ejemplos
- `Ctrl+S` guardar, `Ctrl+Shift+S` guardar todo.
- `Ctrl+Shift+F` formatear documento.
- `Alt+P` abrir panel Problemas.
- `Ctrl+.` aplicar quick fix.

## Errores comunes
- Mezclar atajos del editor y del arbol.
- No usar ayuda contextual (`F1`) cuando hay bloqueo.

## Practica sugerida
- Resuelve un ejercicio usando solo teclado.
