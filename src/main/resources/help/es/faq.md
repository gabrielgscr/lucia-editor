# FAQ y resolucion de problemas

## Objetivo de aprendizaje
- Resolver rapidamente problemas comunes de configuracion y uso.

## Sintaxis clave
```text
Herramientas -> Configuracion
```

## Ejemplos
- Lucia root invalida: configura carpeta que contiene `main.py`.
- Sin diagnosticos: activa diagnosticos en tiempo real.
- Falla run JS: instala Node.js y valida `node` en PATH.

## Errores comunes
- Ejecutable Python mal configurado.
- Comandos ejecutados desde raiz equivocada.

## Practica sugerida
- Crea una checklist de verificacion para estudiantes antes de entregar.
