# Interpolacion de strings

## Objetivo de aprendizaje
- Construir mensajes dinamicos legibles con interpolacion.

## Sintaxis clave
```lucia
let name: string = "Ana";
let total: int = 5;
print("Hola ${name}, total: ${total}");
```

## Ejemplos
- Interpolar variables, expresiones y resultados de metodos.
- Combinar interpolacion con reglas de formato.

## Errores comunes
- Olvidar delimitadores `${...}`.
- Usar concatenacion excesiva en lugar de interpolacion.

## Practica sugerida
- Genera lineas de reporte para una lista de estudiantes.
