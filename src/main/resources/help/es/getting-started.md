# Inicio rapido

## Objetivo de aprendizaje
- Abrir un proyecto Lucia y ejecutar tu primer programa con confianza.

## Sintaxis clave
```lucia
func main() {
    print("Hola Lucia")
}

main()
```

## Ejemplos
- Abrir proyecto: `Archivo -> Abrir proyecto`.
- Crear archivo: `Archivo -> Nuevo archivo Lucia`.
- Guardar: `Ctrl+S`.
- Ejecutar: `F5`.
- Compilar: `F6`.

## Errores comunes
- Ejecutar sin configurar Lucia root valida en configuracion.
- Editar sin guardar antes de run.

## Practica sugerida
- Crea `hola.lucia`, imprime tu nombre y compila a Python y JavaScript.
