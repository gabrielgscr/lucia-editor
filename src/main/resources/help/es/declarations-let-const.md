# Declaraciones con let y const

## Objetivo de aprendizaje
- Declarar valores mutables e inmutables con reglas claras.

## Sintaxis clave
```lucia
let total: int = 0;
const pi: float = 3.1416;
```

## Ejemplos
- Usa `let` para valores que cambian.
- Usa `const` para valores de configuracion fija.

## Errores comunes
- Redeclarar simbolos en el mismo alcance.
- Reasignar una variable `const`.

## Practica sugerida
- Refactoriza un script reemplazando numeros magicos por `const`.
