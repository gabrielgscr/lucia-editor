# Getting started

## Learning objective
- Open a Lucia project and run your first program with confidence.

## Key syntax
```lucia
func main() {
    print("Hello Lucia")
}

main()
```

## Examples
- Open project: `File -> Open project`.
- Create file: `File -> New Lucia file`.
- Save: `Ctrl+S`.
- Run: `F5`.
- Compile: `F6`.

## Common mistakes
- Running without selecting a valid Lucia root in settings.
- Editing without saving before run.

## Suggested practice
- Create `hello.lucia`, print your name, then compile to Python and JavaScript.
