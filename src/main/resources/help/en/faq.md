# FAQ and troubleshooting

## Learning objective
- Resolve common setup and usage issues quickly.

## Key syntax
```text
Tools -> Settings
```

## Examples
- Invalid Lucia root: set folder containing `main.py`.
- Diagnostics not showing: enable real-time diagnostics.
- JavaScript run fails: install Node.js and ensure `node` is in PATH.

## Common mistakes
- Misconfigured Python executable.
- Running commands from wrong project root.

## Suggested practice
- Create a checklist for students before submitting exercises.
